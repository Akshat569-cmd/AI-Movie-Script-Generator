## 📌 Project Overview

An AI-powered movie script generator that uses **LangChain** and **RAG (Retrieval Augmented Generation)** to create creative, structured movie scripts with characters, dialogues and scenes.

---

## ✨ Features

- 🎭 Generates complete movie scripts with characters & dialogues
- 📚 RAG based — uses reference scripts for better output
- 🧠 Powered by LangChain + Groq API (Fast LLM)
- ⚡ FastEmbedding for quick & accurate vector search
- 🎬 Supports multiple genres (Action, Drama, Comedy, etc.)

## 🚀 How to Run
```bash
# Clone the repository
git clone https://github.com/Akshat569-cmd/AI-Movie-Script-Generator.git

# Install dependencies
pip install -r requirements.txt

# Add your Groq API key in .env file
GROQ_API_KEY=your_groq_api_key_here

# Run the app
python app.py
```
